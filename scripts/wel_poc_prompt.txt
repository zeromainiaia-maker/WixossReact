あなたはWixoss TCGの効果テキストを構造化JSONに変換するパーサーです。

## EffectAction 型の概要
```
draw: N                          カードをN枚引く
discard: {count, filter?}        手札を捨てる
banish: {owner, count, filter?}  シグニをバニッシュ（エナへ）
bounce: {owner, count, filter?}  シグニを手札に戻す
down: {owner, count, filter?}    シグニをダウン
up: {owner, type}                シグニ/ルリグをアップ
power: {target, delta, dur?}     パワー修正（dur=turn/perm）
trash: {from, count, filter?}    指定ゾーンからトラッシュへ
energy: {from, count}            エナチャージ
add_hand: {from, count, filter?} 手札に加える
search: {filter, dest, count}    デッキサーチ
freeze: {owner, count}           凍結
grant_keyword: {target, word}    キーワード付与
protect: {target, from[]}        効果耐性

seq: [action...]                 順番に実行
opt: action                      してもよい（optional）
cond: {if, then, else?}          条件付き
choose: {count, options[]}       N択

target/filter:
  owner: self|opp
  count: 1|2|all
  class: "クラス名"
  color: 白|赤|青|緑|黒
  level_gte/lte: N
  power_lte/gte: N
  type: signi|lrig|spell|arts

condition (if):
  field_gte/lte: {owner, val, filter?}
  hand_gte/lte: {owner, val}
  life_gte/lte: {owner, val}
  has_signi: {owner, filter}
  lrig_name_has: "名前"
  opp_used_arts: true
```

## 変換するカード
CardNum: WX02-037
CardName: ＳＰＲＡＳＨ
Type: スペル
EffectText: あなたの＜水獣＞のシグニ１体を対象とし、それをバニッシュする。そうした場合、カードを２枚引く。

## 出力形式
```json
{
  "WX02-037": [
    {
      "effectType": "ACTIVATED",
      "timing": ["MAIN"],
      "action": { ... }
    }
  ]
}
```

JSONのみ出力し、説明文は不要。
